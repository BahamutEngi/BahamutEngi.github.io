数据挖掘(一) 数据质量分析

无标签
发布日期:   2020-08-28
文章字数:   1k

数据质量分析的主要任务是检查原始数据中是否存在脏数据，脏数据一般指不符合要求，以及不能直接进行相应分析的数据。常见的脏数据包括：

  • 缺失值
  • 异常值
  • 不一致的值
  • 重复数据以及包含特殊符号的数据

    1. 缺失值分析

  • 缺失值产生的原因
  1. 有些信息暂时无法获取，或者获取信息的代价太大。
  2. 有些信息被遗漏，包括人为因素和物理故障。
  3. 属性值不存在。缺失值并不一定意味着数据错误，一些对象的某些属性值是不存在的，比如小孩的固定收入。
  • 缺失值的影响
  1. 进行建模时将丢失有用信息。
  2. 模型的不确定性增加。
  3. 有些模型不能处理缺失值。
  • 缺失值的分析

使用统计分析可以得到含有缺失值的属性个数等信息。

2. 异常值（离群点）分析

  • 异常值分析是为了检测数据是否有录入错误或者不合理的数据。
  • 常见的异常值分析方法有简单统计分析、3$\sigma$原则和箱型图分析。

    （1）简单统计分析

    最常用的统计量是最大值和最小值，判断变量中的数据是不是超出了合理的范围。

    （2）3$\sigma$原则

    如果数据服从正态分布，在3$\sigma$原则下，异常值被定义为一组测定值中与平均值的偏差超过三倍标准差的值。

    （3）箱型图分析

    箱型图提供了识别异常值的一个新标准。箱型图依赖实际数据绘制，不要求数据服从何种分布；且判断异常值以四分位数和四分位数间距为基础，具有一定的鲁棒性。

$Q_L$称为下四分位，表示全部观察值中有四分之一的数据取值比它小；

$Q_U$称为上四分位，表示全部观察值中有四分之一的数据取值比它大；

$IQR$称为四分位数间距，是上下四分位数之差，其间包含了所有观察值的一半；

异常值通常定义为小于$Q_L-1.5IQR$或大于$Q_U+1.5IQR$的值

3. 一致性分析

数据不一致是指数据的矛盾性、不相容性。
在数据挖掘的过程中，不一致数据的产生主要发生在数据集成过程中，被挖掘数据来自不同的数据源，对于重新存放的数据未能进行一致性更新。

比如两张表中都存储了用户的电话号码，但在电话号码改变时只更新了一张表的数据，那么这两张表的数据就出现了不一致。

4. 箱型图分析的python实现

数据采用配套例程中的餐饮销量数据，其部分数据如下表所示，其中某些数据存在缺失值。

日期 销量

2015/2/13 3036.8 
import matplotlib.pyplot as plt
import pandas as pd
catering_sale = 'chapter3_catering_sale.xls'
data = pd.read_excel(catering_sale, index_col = u'日期')
plt.rcParams['font.sans-serif'] = ['simhei'] # 用来正常显示中文标签
plt.rcParams['axes.unicode_minus'] = False # 用来正常显示负号

plt.figure()
p = data.boxplot(return_type='dict') #画箱线图，直接使用DataFrame的方法
x = p['fliers'][0].get_xdata() # 'flies'即为异常值的标签
y = p['fliers'][0].get_ydata()
y.sort() #从小到大排序，该方法直接改变原对象

# 用annotate添加注释
# 其中有些相似的点，注解会出现重叠，难以看清，需要一些技巧来控制
# 以下参数都是经过调试的，需要具体问题具体调试
for i in range(len(x)):
    if i > 0:
        plt.annotate(y[i], xy = (x[i], y[i]), xytext=(x[i] + 0.05 -0.8/(y[i] - y[i - 1]), y[i]))
    else:
        plt.annotate(y[i], xy = (x[i], y[i]), xytext=(x[i] + 0.08, y[i]))
plt.show()  # 展示箱型图
